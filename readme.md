## Avionics System 航電系統

### Medium Version
#### System Structure 系統架構
運算核心：Atmega 328 
Developing Environment: Arduino IDE

電路包含可選擇性安裝的感測器及其他外圍設備：
- 氣壓計(I2C)
- 加速度計(I2C)
- 磁力計(I2C)
- SD(SPI)
